# WTF, HTML 그리고 CSS?

**WTF, HTML 그리고 CSS?**는 HTML과 CSS에서 이런 *발' 소리가 나오게 하는 이유의 목록이다. 흔히 혼란스럽게 하는 HTML과 CSS의 진퇴양난, 실수와 딜레마에 관한 목록이다.

**[읽어보기 ☞](http://wtfhtmlcss.com)**
